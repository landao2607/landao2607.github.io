HoverXchange

Duration

July 2020 - November 2020


Role

Co-developer


Skills

HTML, CSS, Javascript

HoverXchange is a web browser extension that converts currency amounts on hover using live exchange rates.

This was my first project doing web development. I was completely new to HTML/CSS and Javascript so it required a lot of self teaching. I was responsible for developing the popup and designing the extension's icon on Safari. It was this project that sparked in me an interest in web development.
